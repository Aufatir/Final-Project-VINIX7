# Final-Project-VINIX7
Sistem prediksi nilai tukar Rupiah (USD/IDR) 30 hari ke depan berbasis Machine Learning menggunakan pendekatan Regression-to-Classification (R2C). Model memprediksi return USD/IDR terlebih dahulu, kemudian mengonversinya menjadi tiga kategori kondisi pasar: Menguat, Stabil, atau Melemah menggunakan asymmetric threshold optimization
